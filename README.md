## 一、宝塔面板是什么，为什么要用它

宝塔面板（BT Panel）是国内最流行的服务器可视化管理工具之一，核心作用就是把原本需要敲命令行才能完成的事情，变成在浏览器里点点按钮就能搞定。

装 LNMP 环境？点一下。创建网站？填个表单。上传文件、配置 SSL、定时任务？都在图形界面里。对于没有 Linux 运维经验的人来说，宝塔面板基本上把服务器管理的门槛降低了 90%。

目前最新正式版为 11.5.0（2026 年 1 月发布），支持主流发行版，包括 Debian 12、Ubuntu 22.04、AlmaLinux 9 等。

## 二、搭建宝塔面板之前，先搞定 VPS

宝塔面板是装在服务器上的软件，所以第一步得有一台 VPS。

选 VPS 主要看这几个维度：

- **单核 CPU 频率**：建站场景对单核性能敏感，频率越高，PHP 解析、数据库查询越快
- **内存**：宝塔面板最低 512MB 能跑，但装了 MySQL 之后容易报错。建议 1GB 起步，跑 WordPress 推荐 2GB
- **机房位置**：访客在哪，机房选哪。国内用户多就选香港或日本节点，欧美用户就选美国或欧洲节点
- **带宽流量**：博客、工具站流量不大，500GB 到 1TB 完全够用
- **价格**：同等性能下，性价比是关键

## 六、宝塔面板 VPS 搭建完整步骤

### 第一步：用 SSH 连接服务器

VPS 部署完成后，你会拿到 IP 地址、root 账号和初始密码。

- **Windows 用户**：推荐 FinalShell（免费，带文件管理）或 Xshell
- **Mac/Linux 用户**：直接在终端输入 `ssh root@你的服务器IP`

连上之后先更新系统，这一步很重要，跳过容易后面报错：

bash
apt update && apt upgrade -y
apt install curl wget -y


等命令跑完，没有报错，就可以进下一步了。

---

### 第二步：安装宝塔面板

宝塔官方提供一键安装脚本，根据你的系统选对应命令。

**Debian / Ubuntu 系统：**

bash
wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && bash install.sh ed8484bec


**CentOS / AlmaLinux 系统：**

bash
wget -O install.sh https://download.bt.cn/install/install_6.0.sh && bash install.sh ed8484bec


运行后会提示确认，输入 `y` 回车，然后等 2-3 分钟。

安装完成后，终端会显示宝塔面板的访问地址、用户名和密码，类似这样：


面板地址：http://your_ip:8888/xxxxxxxx
用户名：xxxxx
密码：xxxxx


**把这些信息复制保存好**，之后登录要用。如果忘了，SSH 里输入 `bt default` 可以重新查看。

---

### 第三步：放行宝塔面板端口

这一步是最多人卡住的地方。

宝塔面板默认用 `8888` 端口，但很多 VPS 的防火墙默认不放行这个端口，导致浏览器访问不了。

**Evoxt 的处理方式**：在 Evoxt 控制台，进入对应 VPS 的管理页面，找到 Firewall（防火墙）功能，添加规则放行 8888 端口。这个操作在控制台图形界面里完成，不需要敲命令，点几下就好。

此外，如果后续要跑网站，还需要同时放行：

- 80（HTTP）
- 443（HTTPS）
- 3306（MySQL，如果需要远程连接）

---

### 第四步：登录宝塔面板并初始化

在浏览器里打开安装完成时显示的面板地址，输入用户名和密码登录。

如果浏览器提示"不安全"，是因为通过 IP 访问没有 SSL 证书，这是正常的，点"高级 → 继续访问"就行。

登录后，宝塔会弹出推荐安装环境的弹窗。建站通常选 **LNMP**（Linux + Nginx + MySQL + PHP）或 **LAMP**（用 Apache 替换 Nginx），一键安装，等待即可。

---

### 第五步：创建网站（以 WordPress 为例）

等 LNMP 环境装好之后，在宝塔左边菜单点"网站 → 添加站点"，填入域名（域名需要提前解析到服务器 IP），选择 PHP 版本，宝塔会自动建好目录和数据库。

然后在 WordPress 官网下载压缩包，通过宝塔的文件管理上传解压，浏览器访问域名就进入 WordPress 安装向导了。

整个流程跑下来，从买 VPS 到网站上线，一两个小时内能搞定。

## 七、常见报错与解决方法

**Q：面板地址打开了，但显示无法连接？**  
A：先检查 VPS 防火墙有没有放行 8888 端口。Evoxt 控制台里的 Firewall 功能可以直接处理，不用 SSH 进去操作。

**Q：安装宝塔时提示 curl: command not found？**  
A：Debian 12 精简版默认不带 curl，先运行 `apt install curl wget -y` 装上再执行安装命令。

**Q：安装完宝塔，MySQL 启动报错？**  
A：通常是内存不足。512MB 内存装完宝塔本身后几乎没有剩余。建议使用 1GB 以上内存的套餐，VM-1（2GB 内存）最为合适。

**Q：忘了宝塔面板的登录地址和密码？**  
A：SSH 连接服务器后输入 `bt default` 即可重新显示。
